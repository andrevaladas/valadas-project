O que é e como funciona o SGTT

O SGTT - Sistema de Garantia Total Tipler é o sistema de garantia de recapagens oferecido por Borrachas Tipler Ltda. aos seus concessionários.

Seu objetivo, é permitir o gerenciamento do processo de garantia de recapagens.
O sistema tem tres tipos de usuários a saber:

  • Usuário Tipler
    • Responsável por manter as tabelas básicas do sistema e gerenciar as contas correntes dos concessionários.
  • Usuário Concessionário
    • Responsável por registrar as recapagens garantidas conforme critérios pré-estabelecidos e indenizar o cliente final em caso de necessidade.
  • Usuário Inspetor
    • Responsável por validar as indenizações concedidas pelos concessionários as clientes finais.

Clique sobre um dos tipos de usuário acima para acessar o help específico.

Consideração:
Cada um dos tipos de usuários tem visões diferentes do sistema de modo a permitir que suas tarefas possam ser efetuadas a contento.
Assim, o menu de acesso as telas é necessariamente diferente entre os tipos de usuários visto que cada um tem objetivos distintos no uso do sistema.
