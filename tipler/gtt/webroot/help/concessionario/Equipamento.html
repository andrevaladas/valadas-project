Equipamento necessário para executar o sistema

Para usar o sistema é necessário que se disponha de um microcomputador compatível com IBM PC com:

- 1GB de memória ou superior
- Disco rígido ( winchester) com 1GB disponível
- Unidade de CD-ROM
 - Sistema operacional Windows XP ou superior
- Browser IE 7 ou superior, Firefox 2 ou superior
- Flash player 7 ou superior
- Impressora para emissão de relatórios ( Ink Jet )
- Monitor
- Teclado
- Mouse
